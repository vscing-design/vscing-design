## 包管理工具
- npx 命令执行工具
- 先驱者 npm 属于node的一个管理工具
- 创新者 yarn 解决了npm依赖下载慢、npm是按照队列依次执行，yarn是同步执行。monorepo 支持、缓存安装、离线下载、文件锁（Locking）。
- 更高效 pnpm 主要推出了[依赖软链接](https://pnpm.io/next/symlinked-node-modules-structure)，解决文件复用等问题。
